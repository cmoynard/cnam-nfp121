1 Installation de subversion (SVN)

1.1 Pour linux

Utiliser le sytème de gestion de paquet de votre système pour installer le paquet subsersion. Par exemple, sous ubuntu on peut faire :

$ sudo apt-get install subversion

1.2 Pour MacOSX

Utiliser un système de gestion de paquets tel que homebrew :

$ brew install subversion

1.3 Pour Windows

On peut suivre les indications données sur StackOverflow : Command-line svn for Windows?.

1.4 Interface graphique pour SVN

SmartSVN, qui ne semple plus être maintenu depuis 2016, propose une interface graphique pour accéder à un dépôt SVN. Il est disponible pour Linux, MacOSX et Windows.

Pour Windows, vous pouvez bien sûr utiliser TortoiseSVN.
